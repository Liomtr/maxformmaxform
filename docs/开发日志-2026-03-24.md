### 3. 打通问卷结果页真实统计

修改文件：

- `backend/src/routes/surveys.js`
- `backend/test/routes.test.js`
- `frontend/src/api/surveys.ts`
- `frontend/src/views/survey/ResultsPage.vue`

- 题目分析已能展示真实的：
  - 单选 / 多选 / 排序分布
  - 文本题样本与时间范围
  - 滑动条最小值 / 最大值 / 平均值
  - 上传题文件样本与文件数

结果：

- 结果页从“只有总提交数”推进到“汇总指标 + 题目级统计 + 系统环境统计”的真实闭环。
- 后端补充了对应自动化测试，保证统计输出结构可回归验证。

### 4. 接通结果页到答卷管理与导出链路

修改文件：

- `frontend/src/views/survey/SurveyAnswersPanel.vue`
- `frontend/src/views/survey/AnswerManagementPage.vue`
- `frontend/src/api/surveyAnswers.ts`

本次推进内容：

- 将结果页中的“查看答卷”按钮接到 `/surveys/answers` 页面，并携带 `surveyId` 与问卷标题。
- 将结果页中的“导出数据”按钮接到现有 `/answers/download/survey` 接口，直接下载 Excel。
- 重写 `AnswerManagementPage.vue`，从输入框占位页推进为可用答卷管理页：
  - 自动读取路由中的 `surveyId`
  - 分页查看答卷
  - 展示提交时间、状态、IP、耗时、答案预览
  - 支持单条删除
  - 支持 Excel 导出
- “下载附件”按钮暂时保留为显式待办提示，因为后端尚无批量打包下载接口。

结果：

- 结果页与答卷管理页已形成实际业务链路，不再停留在 alert 占位。
- 用户可以从结果页直接进入答卷列表，或直接导出当前问卷答卷。
